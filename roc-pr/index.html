Comment ça marche : chaque modèle attribue un score continu ∈ [0, 1] à chaque observation. Les scores sont générés via deux gaussiennes (positifs vs négatifs) séparées par un paramètre de qualité. On balaie les seuils pour calculer TPR, FPR, Precision, Recall. La simulation Monte Carlo répète l'expérience N fois pour montrer la variabilité. Observe comment la courbe PR est beaucoup plus sensible au déséquilibre que la ROC.
